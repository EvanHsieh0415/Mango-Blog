## 說明

此資料包新增了羊毛與其製品的各種染色配方，可將染色過的製品再次染色  
也增加了羊毛製作成線的配方

**支援的方塊有：**
| 標籤 | 中文名 |
| ----------------------- | ------ |
| `wool_tweaks:wt_wool` | 羊毛 |
| `wool_tweaks:wt_carpet` | 地毯 |
| `wool_tweaks:wt_bed` | 床 |
