## 配方類型

### 批次喚魂

相關方塊：

- `create:encased_fan`
- `#create:fan_processing_catalysts/haunting`

```json
{
  "type": "create:haunting",
  "ingredients:": Ingredients<Item>[],
  "result": Result<Item>[]
}
```

:::details 範例

```json
{
  "type": "create:haunting",
  "ingredients": [
    {
      "tag": "forge:cobblestone"
    }
  ],
  "results": [
    {
      "item": "minecraft:blackstone"
    }
  ]
}
```

:::

## 資料類型

### object&lt;`Ingredient`&gt;

- `nbt`: 物件 = {}（可選）  
  NBT 篩選器
- `chance`: 數字 = 1（可選）

#### &lt;Item&gt;

- `item|tag`: 字串  
  物品 ID 或標籤，必須是資源路徑（Resource location）

#### &lt;Block&gt;

- `item|tag`: string  
  方塊 ID 或標籤，必須是資源路徑（Resource location）

#### &lt;Fluid&gt;

- `fluid|fluidTag`: string  
  流體 ID 或標籤，必須是資源路徑（Resource location）
- `amount`: number = 1（可選）  
  流體數量

### object&lt;`Result`&gt;

- `nbt`: object = {}（可選）  
  NBT 篩選器
- `chance`: number = 1（可選）  
  成品產出機率

#### &lt;Item&gt;

- `item`: string  
  物品 ID 或標籤，必須是資源路徑（Resource location）
- `count`: number = 1（可選）  
  物品數量

#### &lt;Block&gt;

- `item|tag`: string  
  方塊 ID 或標籤，必須是資源路徑（Resource location）

#### &lt;Fluid&gt;

- `fluid`: string  
  流體 ID 或標籤，必須是資源路徑（Resource location）
- `amount`: number = 1（可選）  
  流體數量

### number&lt;`GameTick`&gt;

20 刻 = 1 秒  
[刻－中文 Minecraft Wiki](https://zh.minecraft.wiki/w/%E5%88%BB)
